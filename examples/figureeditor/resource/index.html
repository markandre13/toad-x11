Fischland

Fischland is a 2d vector graphics editor and presentation program.

ATTENTION! Fischland is still under construction. Please check out for more recent version at http://www.mark13.org/fischland/ and send anything you want to say about this program to <mhopf@mark13.org>.

Select Tool
  • click
    Select and move figures and their handles.
    When there's no figure under the mouse pointer, starts to select an area when the mouse is moved.
  • SHIFT+click
    Select an area.
  • CTRL+click
    Select/deselect single figures.
  • SHIFT+CTRL+click
    Select an additional area.
  • double click
    Some figures will also react to a double click, like text figures, to enter a special editing mode.
  • right click
    Some figures will open a popup menu when clicked on with the right mouse button. Eg. polygons and bezier will allow you to add and delete points.

When moving the handles of beziers you may also use
  • SHIFT
    For smooth symmetrical corners with related handles being on a line and of same distance.
  • CTRL
    For smooth corners with related handles being on a straight line but with different distance.

Insert a line, a polygon, a bezier line or a filled bezier.

Add points to the bezier with a click of the left mouse button and add the last point with a double click or the enter key.

The backspace key will remove the last point added to the bezier.

A right click on a finished bezier figure will open a popup menu with additional options like:

  • insert point
  • delete point
  • ...

The escape key [ESC] will abort the creation of a figure or deselect a figure after creation.
